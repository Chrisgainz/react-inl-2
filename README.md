# react inlämningsuppgift 2 grupp 7

- Projektbeskrivning: Vi har gjort en app där man kan gå in och titta på olika recept på måltider.
  Man kan exempelvis spara måltider som favoriter och skapa egna måltider.
  Vi använder oss av https://www.themealdb.com/api.php som api för hämtning av data.

- Installation:

1. Klona repot: git clone <repo-url>
2. Installera dependencies: npm install
3. Starta projektet: npm run dev

- Tech stack: 

React, Vite, React router, Axios, css modules, Javascript, api (TheMealDB)
